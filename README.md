# construction-materials-web

Веб-версия приложения для заявок на строительные материалы (аналог `construction-materials`).

## Запуск

```bash
npm install
npm run dev
```

Сайт откроется на http://localhost:5176

## Возможности

- выбор региона (СПБ, МСК, КРД)
- каталог с ценами по региону, фильтр по категории и поиск
- оформление заказа с подтверждением
- при отказе — контрпредложение (дешёвый аналог или −5%)
- скачивание JSON-заявки (как в консольной версии)

## Сборка

```bash
npm run build
npm run preview
```
